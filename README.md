# 購物網站練習

這是一個使用 **Vue.js** 作為前端、**Node.js (Express)** 作為後端的全棧購物網站練習專案。

## 使用技術棧

- **前端:** Vue 3 (Vite), Vue Router, Pinia
- **後端:** Node.js, Express
- **資料庫:** MySQL
- **版本控制:** Git / GitHub

## 專案結構說明

- `/frontend`: Vue 3 前端專案
- `/backend`: Node.js Express 後端 API 伺服器

## 啟動步驟

- **後端啟動**

  ```
  cd backend
  npm install
  node index.js
  ```
